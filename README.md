# My-first-project-



Prime Number Checker 🧮

📌 Description

This is a simple Python program to check whether a given number is a prime number or not.

---

🚀 Features

- Takes user input
- Checks prime condition
- Displays result clearly

---

🛠️ Requirements

- Python 3.x

---

▶️ How to Run

1. Clone this repository:

git clone https://github.com/your-username/prime-checker.git

2. Navigate to the folder:

cd prime-checker

3. Run the program:

python main.py

---

📷 Example Output

Enter a number: 7
7 is a Prime Number

---

📚 Logic Used

A number is prime if it is divisible only by 1 and itself.
